# Partial Application

여러개의 인자를 받는 함수가 있을 때 일부 인자를 고정한 함수를 만드는 기법

```javascript
var plus = function(a, b, c) {
    return a + b + c
}
```

이를 [[currying]]하면, 다음과 같다.

```javascript
// Partial application
const add = x => (y, z) => x + y + z;
// currying
const add = x => y => z => x + y + z;
```

공통점:  
[[currying]]과 [[partial-application]] 함수들이 받는 인자는 기존 함수가 받는 파라미터보다 최소 1개 이상 덜 받는다.

차이점:  
[[currying]]은 단 한개만, [[partial-application]]은 여러개의 파라미터를 받을 수 있다.

## 왜 필요한가

모두 lazy evaluation이 가능하다.

```javascript
const getRGB = r => g => b => `rgb(${r},${g},${b})`;

// `rgb(0,0,${b})`
const getBlue = getRGB(0)(0); 
const div = getElementById('myDiv');

div.addEventListener('click', () => {
  const randomColor = Math.floor(Math.random() * 255) + 0;
  div.style.background = getBlue(randomColor);
});
```
