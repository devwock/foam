# Pure Function

순수 함수는 메모리나 I/O로부터 side effect를 갖지 않음.

- 결과가 사용되지 않을 경우 다른 표현식에 영향을 주지 않고 제거 가능
- 순수한 함수는 부작용을 발생시키지 않는 인자를 호출할 떄, 결과 값은 인자 리스트에 관련한 상수이다. 순수한 함수가 같은 인자를 호출한다면 같은 결과를 받을 것이다.
- 두개의 순수한 표현식 사이에 데이터 의존성이 없는 경우, 순서를 반대로 하거나 병렬로 수행될 수 있고, 서로 간섭을 할 수 없다. (thread-safe)
- 모든 언어가 부작용을 허용하지 않는 경우, 어떠한 평가 전략이 사용된다. 프로그램 내의 순서를 변경하거나 식 계산을 결합을 위한 자유를 컴파일러에게 제공한다. (deforestation)
