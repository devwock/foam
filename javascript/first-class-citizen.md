# First Class Citizen

자바스크립트는 함수를 [[first-class citizen]]으로 대해준다. 자바스크립트나 다른 함수형 프로그래밍 언어 함수들은 전부 객체이기 때문.

자바스크립트에서 함수는 `function` 객체이다.  
함수에 속성을 추가할 수도 있다.

```javascript
function test() {
    console.log('hello, world');
}
test.test = 'hello';
console.log(test.test);
```

위 문법은 유효하나 매우 위험하게 여겨진다. 필요한 경우 오브젝트를 사용하는것이 권장된다.

함수를 변수에 할당하기

```javascript
const square = function(x) {
    return x * x
}
square(5);
const foo = square;
foo(6);
```

함수를 파라미터로 넘기기

```javascript
function formalGreeting() {
    console.log("how are your?");
}
function casualGreeting() {
    console.log("What's up?")
}
function greet(type, greeeFormal, greetCasual) {
    if (type === 'formal') {
        greetFormal();
    } else if (type === 'casual') {
        greetCasual()
    }
}
greet('casual', formalGreeting, casualGreeting);
```
