# Immediately Invoked Function Expression

즉시 실행 함수의 줄임말. 정의와 동시에 즉시 실행된다.

## 즉시 실행 함수 표현식

[[함수 리터럴]]을 `()`로 감싸 바로 실행시킨다.

```javascript
(function() { console.log('Hello, World!') })();
```

즉시 평가하여 실행하기 때문에 스코프를 오염시키지 않는다.

### 다양한 표현식들

```javascript
function() {}
```

선언을 이렇게 하면 파서는 함수 선언문으로 인지. 해석기에 명령을 지시하고 사라짐.  
그래서 `()`로 묶어주어 `함수 표현식`으로 명시적으로 선언.  
그 외 연산자를 앞에 붙일 경우도 가능하다.
