# 함수 리터럴

자바스크립트에서 함수를 정의하는 표현식을 함수 리터럴이라 한다. 
다음 4 요소로 구성된다.

1. 예약어 function (필수)
2. 함수 이름 (선택)
3. 매개변수 집합 (필수)
4. 함수 본문 (필수)

```javascript
function add(a, b) { // 1. function, 2. 이름 add, 3. 매개변수 집합 (a, b)
    return a + b // 함수 본문
}
```

함수 이름을 선택사항으로 사용하려면 다음 조건을 충족해야 한다.

1. 함수를 할당받을 변수를 지정
2. 함수를 즉시 호출 [[iife]]

```javascript
const add = function(a, b) { return a + b };
(function(a, b) { return a + b}) (1, 2);
```
